# Exact Diagonalization of a Quantum Hamiltonian

Implementation of a quantum many-body solver using exact diagonalization.

## Overview

This project models a quantum system on a one-dimensional lattice and computes its energy spectrum by constructing and diagonalizing the Hamiltonian matrix.

The implementation includes:
- Binary state-space representation
- Hamiltonian matrix construction
- Jacobi diagonalization method
- Eigenvalue extraction
- Internal energy computation across temperatures

## Technical Highlights

- Implemented in C
- Custom numerical solver
- Manual state-space construction
- Physics-to-code pipeline from model to thermodynamic results
